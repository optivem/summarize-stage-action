# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [Unreleased]

### Added
- Initial release of Summarize Stage Action
- Support for customizable emojis for different stage results
- Custom success message and content support
- Markdown support for rich content formatting
- Composite action implementation for cross-platform compatibility

### Features
- Generate summaries for success, failure, cancelled, and unknown stage states
- Configurable emojis for each stage result type
- Custom success message and content with full markdown support
- Automatic GitHub Step Summary integration
- Simple composite action requiring no external dependencies

## [1.0.0] - 2024-10-13

### Added
- Initial public release
- Core functionality for stage summarization
- Comprehensive documentation and examples
- GitHub Marketplace publishing readiness

### Security
- No external dependencies to minimize security risks
- Pure bash implementation for transparency
